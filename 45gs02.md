# 45GS02 Implementation Task List

This document tracks the progress toward a complete 45GS02 CPU core for the MEGA65.

## 1. Instruction Set
- [x] Baseline 6502 Instructions (Core set: ADC, SBC, LDA, STA, etc.)
- [x] 65C02 Extensions (INA, DEA, PHX/PLX, PHY/PLY, BRA)
- [x] Basic 45GS02/65CE02 Transfers (TAZ, TZA, TAB, TBA)
- [x] Basic 65CE02 Stack (PHZ, PLZ)
- [x] 16-bit Arithmetic/Increment (ASW, ROW, DEW, INW)
- [x] Word Push/Pull (PHW)
- [x] Bit Manipulation (BBR/BBS, RMB/SMB)
- [x] Advanced Flow Control (BSR, RTN)
- [x] MEGA65 Specifics (EOM, NEG, MAP)
- [x] 32-bit Quad operations (LDQ, STQ, ADCQ, SBCQ, CMPQ, ANDQ, ORAQ, EORQ, ASLQ, LSRQ, ROLQ, RORQ, ASRQ, NEGQ, INQ, DEQ, BITQ — all addressing modes including #imm32)

## 2. Addressing Modes
- [x] Standard 6502 Modes (Immediate, Absolute, Zero Page, Indexed, Indirect)
- [x] Base Page Indirect (Respecting the `B` register for all ZP operations)
- [x] Stack Relative Indirect (`(d,S),Y`) — LDA and STA
- [x] 32-bit Indirect Pointers (Full 28-bit space access via `$EA` prefix)
- [x] Indirect Indexed Z-register (`(zp),Z` and `[$zp],Z`)

## 3. MMU & Memory Mapping (`MAP`)
- [x] Functional `translate()` helper for address resolution
- [x] 8-block Mapping Logic (8 x 8KB blocks with per-block offsets)
- [ ] Hardware Register Integration (Mapping registers in `$D000`/`$D600` range)
- [x] 28-bit address space (via MapMmu + SparseMemoryBus)

## 4. Stack Pointer Logic
- [x] Basic 16-bit SP storage
- [x] 8-bit / 16-bit Stack Mode switching (CLE/SEE toggle FLAG_E; push/pull check it)
- [ ] B-register offset integration for stack operations

## 5. System Integration & Interrupts
- [x] Basic IRQ/NMI/Reset handling
- [ ] Interrupt Masking (Status register extensions)
- [ ] Vector Redirection via MMU
- [x] Hypervisor Exit Traps (`$D6CF` trigger)

## 6. Performance & Accuracy
- [x] Basic Fetch-Decode-Execute cycle
- [ ] Cycle-accurate timing for all opcodes/modes
- [ ] I/O Stalls and DMA contention simulation
- [x] Decimal Mode (`SED`) accuracy (BCD arithmetic in ADC/SBC)

## 7. Toolchain & Debugging
- [x] Full 45GS02 instruction disassembly (all opcodes, quad prefix, 32-bit indirect prefix)
- [x] Register inspection in CLI
- [ ] Symbol table integration for 28-bit addresses

**Testing Note**: The validation suite (`tests/45gs02/validate.py`) cross-checks the implementation against `xemu-xmega65`.
*Note: Running validation may trigger a GUI dialog from xemu even in headless mode.*
